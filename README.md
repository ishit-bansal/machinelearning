# Machine Learning Made Visual

An interactive educational website that teaches machine learning concepts through stunning visuals, animations, and hands-on demos.

## Technologies

- **Vite** - Build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI framework
- **shadcn-ui** - Component library
- **Tailwind CSS** - Styling
- **TensorFlow.js** - Machine learning in the browser
- **Framer Motion** - Animations

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- **Interactive Digit Recognizer** - Draw digits and see AI predictions in real-time
- **Visual Learning** - Animated explanations of ML concepts
- **Real-World Examples** - See how ML is used in everyday applications
- **Pattern Recognition** - Understand how machines learn from data
